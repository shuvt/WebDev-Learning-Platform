# WebDevLearn — обучающая платформа по веб-разработке
 
Веб-платформа для изучения PHP и SQL, где теория и практика находятся в одном месте.  
Развёрнута на [webdevlearn.ru](https://webdevlearn.ru)
 
---
 
## Возможности
 
-  **4 теоретических курса** — SQL, PHP, HTML+CSS, пошаговое создание веб-приложения «Библиотека»
-  **SQL-тренажёр** с автоматической проверкой запросов на реальной базе Firebird Employee
-  **Редактирование контента** — преподаватель правит темы прямо в браузере через редактор Quill
-  **Отслеживание прогресса** по теоретическим курсам и практическим заданиям
-  **Система ролей** — студент, преподаватель, администратор
